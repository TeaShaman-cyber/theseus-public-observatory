# Data

This directory contains public-source snapshots generated by GitHub Actions or
local runs.

Layout:

```text
data/YYYY-MM-DD/public-status.jsonl
data/latest/public-status.json
```

Each JSONL row is one collection run with:

- `collected_at`
- `sources[]`
- compact source summaries
- fetch status and latency

Only public data belongs here.
